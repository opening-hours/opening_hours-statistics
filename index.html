opening_hours statistics

How many amenities are open for a given time?

Shows the percentage of open amenities (based on all opening_hours in OSM) for a given time of the week.

What is in the database?

The purpose of this statistic is to show the data quality and growth over time. For more information, check out the Blog entry by ypid.

For the internals check out the README on GitHub.
